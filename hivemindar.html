HivemindAR

HivemindAR is a augmented reality(AR) mobile application that simulates the painting on the wall using a smart phone. This project was created during our internship in Mr. Geek Philippines, Inc. known for specializing in Web and Mobile applications and offer innovative products.

We used Unity Real-Time as our engine in making the HivemindAR mobile application. We also used AR Foundation for the framework and Github for tracking the team's progress, and this made it easier for us to share code files during the development.
